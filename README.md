# python3
:penguin: Default Python 3 environment managed by Poetry

## Installation

```shell
$ git clone https://github.com/astropenguin/python3.git
$ cd python3
$ poetry install
```

## Usage

```shell
$ cd python3
$ poetry run jupyter lab
```

## Add packages

```shell
$ cd python3
$ poetry add <package> ...
```

## Delete an environment

```shell
$ cd python3
$ poetry env remove 3.7
```
